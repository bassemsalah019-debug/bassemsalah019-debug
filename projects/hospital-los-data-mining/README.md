## 📋 Overview

Two-part project: (1) Production-grade hospital management DB design in SQL Server + (2) Full EDA & data mining pipeline on 318,438 real hospital admission records.

## 🏆 Key Numbers

- 318,438 patient records · 32 hospitals · 11 LOS classes
- 22-table SQL Server DB · 28 FK · 6 procedures · 3 triggers
- 10 SQL analytical views · 6 RBAC roles · 21 indexes
- 20 EDA questions answered · Power BI 5-page dashboard

## 🔑 Key Insight

Patient Visitors is the strongest LOS predictor (mean rises 2.6 → 7.9 as stay increases).
